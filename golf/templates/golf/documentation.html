General documentation

Python is a development language that is simple to read. It is an interpretted and compiled language, so it is fast and yet very helpful when it comes to debugging. Installed on the system using a Windows installer or apt-get install python

django (JANG-goh) is the web framework selected for this project because it gets us running very quickly. It also manages our data for us through the model.py file. Installed on the system with pip install django

SQLite is a datastore. Contained in a single file, data can be viewed with sqlite edittor, but shouldn't be required because django is managing this for us.

bootstrap is a front-end web framework which takes all of the effort out of standard reusable components. Not installed, is loaded from base.html from bootstrap cdn

Font Awesome is a font and icon framework. Not using a ton, yet. Not installed, is loaded from base.html from bootstrap cdn

jQuery is a javascript library that creates a simple interface to the document object model (DOM) of a webpage. Much faster and cross platform. Should be used for all javascripting. Not installed, is loaded from jquery cdn

There are several other libraries loaded, but they are for very specific components such as datetimepicker and quicksearch.
